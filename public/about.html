📸PosePix - Photobooth Web App

Photobooth is an interactive web application that allows users to capture multiple poses directly from their webcam. It simulates a real photobooth experience — complete with timers, continuous shots, stickers, and retake options — making it fun and user-friendly for both events and casual use.

🚀 Features

  • Capture live photos using your webcam via WebRTC
  • Set a timer and capture multiple poses automatically
  • Double-click on a photo to retake it instantly
  • Add and remove stickers to customize your pose card
  • Upload and store photos securely using Cloudinary API and MongoDB
  • Beautiful, responsive UI designed with Tailwind CSS
